# LDAP / Active Directory Tools for Bitrix24

Набор PHP-скриптов для работы с LDAP/Active Directory в среде Битрикс24.

## 📁 Файлы

### 1. `ldaptest.php`
Инструмент для тестирования подключения и выборок с AD-сервера.

**Возможности:**
- Тестирование подключения к LDAP
- Просмотр доступных Base DN
- Выполнение тестовых запросов с настраиваемыми фильтрами
- Преднастроенные фильтры для пользователей и групп

**Использование:**
Откройте файл в браузере и заполните параметры подключения:
- LDAP сервер
- Порт (по умолчанию 389)
- Base DN
- Логин/пароль для привязки

---

### 2. `ad_import_diagnostic.php`
Диагностический инструмент для импорта данных из Active Directory.

**Возможности:**
- Проверка подключения к AD
- Настройка фильтров LDAP
- Выбор атрибутов для импорта
- Диагностика проблем подключения
- Предпросмотр данных перед импортом

**Параметры по умолчанию:**
- Порт: `389`
- Фильтр: `(&(objectClass=user)(objectCategory=person))`
- Поля: `distinguishedName,sAMAccountName,userPrincipalName,givenName,sn,displayName,mail,department,title,company,manager,userAccountControl,whenCreated,whenChanged`

---

### 3. `export_company_structure.php`
Скрипт для экспорта структуры компании Битрикс24 в графическом формате.

**Возможности:**
- Экспорт в SVG
- Экспорт в JPG
- Предпросмотр в HTML

**Использование:**
```
/pub/company/export_company_structure.php?format=svg
/pub/company/export_company_structure.php?format=jpg
/pub/company/export_company_structure.php?format=html
```

**Особенности:**
- Отрисовка организационной структуры без сотрудников
- Настраиваемые параметры визуализации
- Автоматическое построение иерархии

---

## ⚙️ Требования

- PHP 7.0+
- Расширение `ldap`
- Битрикс24 (для `export_company_structure.php`)
- Доступ к LDAP/Active Directory серверу

## 🔧 Установка

1. Разместите файлы в директории вашего сайта Битрикс24
2. Убедитесь, что установлено расширение `php-ldap`
3. Настройте параметры подключения к AD в каждом скрипте
4. Для `export_company_structure.php` убедитесь, что подключён модуль `iblock`

## 🔐 Безопасность

⚠️ **Внимание:** Эти скрипты содержат чувствительные настройки подключения к LDAP/AD.

- Не размещайте файлы в публично доступных директориях
- Используйте HTTPS
- Ограничьте доступ по IP или авторизации
- Не храните пароли в открытом виде в продакшене
